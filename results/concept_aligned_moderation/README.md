# Concept-Aligned Moderation Results

These tables test platform membership as a moderator of marketing-mix
effectiveness on capacity utilisation.

Main files:

- `primary_ur_moderation_terms.csv`: primary `ur_boxcox` moderation terms for
  the all-sample market-FE and unit-FE models.
- `moderation_synthesis.csv`: robustness synthesis across outcomes, samples,
  platform types, and fixed-effect strategies.
- `concept_moderation_terms.csv`: full long-form coefficient table.
- `concept_model_sample_summary.csv`: sample diagnostics.
