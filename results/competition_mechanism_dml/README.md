# Competition Mechanism and DML Results

These tables test whether competitive intensity strengthens the relationship
between platform membership and pricing-friction effects.

Main files:

- `competition_mechanism_terms.csv`: full long-form coefficient table.
- `competition_mechanism_synthesis.csv`: summary of triple-interaction terms.
- `competition_mechanism_sample_summary.csv`: sample diagnostics.

Interpretation: this evidence is useful as a mechanism probe, but the
competition triple-interactions are not yet stable enough to anchor the main
paper claim.
