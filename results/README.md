# Curated Results

This folder contains only the result tables used for the current paper story.
Earlier exploratory FreeNow-only diagnostics and classic DiD outputs are
intentionally excluded from the clean module.

## Included

- `concept_aligned_moderation/`: main moderation models aligned with the concept paper.
- `competition_mechanism_dml/`: exploratory competition mechanism and DML robustness.
- `platform_typology/`: compact platform-type summary and crosswalk.

## Excluded

- FreeNow-only sequential diagnostics.
- Classic FreeNow DiD diagnostics.
- Older broad platform model-family outputs.
